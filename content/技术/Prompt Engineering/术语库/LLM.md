# LLM

**LLM（Large Language Model，大语言模型）**：以海量文本训练出来、能够进行自然语言理解与生成的模型（对话、写作、翻译、推理、代码生成等）。

## 缩写 / 全称 / 同义词

- **缩写**：LLM
- **全称**：Large Language Model
- **同义词/近义**：大模型（语境相关，可能也包含多模态模型）
